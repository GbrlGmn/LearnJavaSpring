Mini Geladeria - Estudo de MVC com Spring

Este projeto foi desenvolvido como exercício prático para aprender e aplicar o padrão *MVC (Model-View-Controller)* utilizando Java e Spring Framework.

A proposta é simular uma mini geladeria, permitindo gerenciar sabores e produtos, organizando o código nas camadas de:

Model → Representação das entidades
View → Interface de exibição
Controller → Regras e controle de fluxo

O foco principal do projeto é aprendizado e prática de arquitetura em aplicações web.
